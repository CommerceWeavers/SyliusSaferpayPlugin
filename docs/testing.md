## Testing

### Run PHPSpec tests

```bash
vendor/bin/phpspec run
```

### Run PHPUnit tests

```bash
vendor/bin/phpunit
```

### Run Behat scenarios

```bash
vendor/bin/behat --strict
```

### Run Static Analysis checks

```bash
vendor/bin/psalm
vendor/bin/phpstan analyse
```

### Run Coding Standard check

```bash
vendor/bin/ecs check
```
